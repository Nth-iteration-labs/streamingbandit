Welcome

Hi! Clearly, you have gotten an instance of StreamingBandit up and running! Nice.

However, StreamingBandit is a REST application and it does not have a nice UI; at least, not one that's included in the default git repository (which, in case you are wondering, can be found here. Also, the streamingbandit docker container, in case that's what you used to get started, does not include a front end.

There are however a number of ways to get started:

  • If you just want to use the REST API itself, and interact with it using your own application or front-end, a good place to start is the documentation. Here you will find descriptions of all the REST endpoints and a number of usage examples.
  • If you want to use a convenient front-end, please check out our front-end repository. The README on github should be enough to get you started and hence proceed using StreamingBandit but now with a nice UI, buttons, and all that. You can also get the front end up and running by using its docker container.

If you are wondering what this software is good for (which, admittedly, is a bit late after you got it up and running) we recommend you start by reading https://arxiv.org/abs/1602.06700/.

In case you have any quesitons please do contact us by sending an email to m.c.kaptein@uvt.nl. If you find any issues, bugs, or have feature requests, than please use our github page.
